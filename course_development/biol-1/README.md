# BIOL-1: Biology 1 at Pelican Bay Prison

## Course Overview

Biology 1 course materials for Spring 2026 at Pelican Bay Prison.

## Course Structure

This course is organized into modules, each containing assignments, lecture materials, lab protocols, study guides, and quizzes.

### Syllabus

The course syllabus is available in the [`syllabus/`](syllabus/) directory. The syllabus is processed to multiple formats (PDF, MP3, DOCX, HTML, TXT) for accessibility.

### Course Materials

Navigate to the [`course/`](course/) directory to access public course materials organized by module.

### Private Materials

Instructor-only materials are available in the [`private/`](private/) directory.

### Resources
